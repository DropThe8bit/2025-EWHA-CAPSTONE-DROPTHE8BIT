# Project-Summary

| 항목 | 내용 |

| (1) 문제 정의 | AI 기술이 아이의 창의력에 어떤 도움을 줄 수 있을지 고민한 끝에 이미지 보정 AI 등을 활용해 아이의 그림을 캐릭터로 변환하고, 부모와 함께 동화를 만들어가는 서비스를 개발하고자 한다.<br><br>연구에 따르면, 5-8세 아동은 하루 평균 약 3시간을 기기를 통한 미디어 소비에 사용하며, 그 대부분을 영상 시청에 할애하고 있다. 그러나 수동적 영상 시청을 할 경우 정서 표현 수준이 낮았던 반면, 인터랙션 활동을 결합했을 때 정서 어휘 사용이 약 45% 더 높아지는 것으로 나타났다.<br><br>이러한 결과는 단순한 미디어 소비가 아닌, 아이가 사고력과 창의력을 마음껏 표현할 수 있는 활동이 필요함을 보여준다. 따라서 우리는 이러한 활동을 AI 기반 동화책 제작으로 구현하여, 아이가 능동적으로 참여하고 창의성을 발휘할 수 있도록 하고자 한다.<br><br>그러나 현재 부모와 자녀 간의 정서적 교감을 증진시키면서, 동시에 가정과 유치원 모두에서 관리·활용할 수 있는 동화책 제작 서비스는 부족하다. 이에 따라 AI를 활용해 창의력 발달, 정서적 교감, 교육 현장과 가정 연계성을 아우를 수 있는 새로운 서비스의 필요성을 느끼고 있다.<br><br>**Target Customer 및 Pain points**<br>본 서비스의 주요 대상 고객은 다음과 같다.<br><br>1. **4~8세 아동**<br>- 자신의 생각과 감정을 직접 표현할 기회가 부족한 아이<br>- 상상력을 키울 수 있는 능동적 활동이 제한적인 아이<br><br>2. **맞벌이 부모(양육자)**<br>- 바쁜 일정으로 자녀에게 충분한 교육적 케어를 제공하기 어려움<br>- 영상 콘텐츠에 의존하는 양육 방식에 한계를 느끼는 부모<br><br>3. **국공립 유치원 교사**<br>- 다양한 성향의 아이들을 만족시킬 수 있는 콘텐츠 부족<br>- 가정과 연계하여 활용할 교육 자료가 부족<br><br>**다음과 같은 문제를 해결하여 창의력을 자극하는 교육 도메인을 만드는 것을 목표로 한다.**<br>1. 아이의 적극적인 참여 유도<br>2. 맞벌이 부모와 아이의 정서적 교감 증진<br>3. 가정 연계 콘텐츠 활용 |  |
| (2) 기존연구와의 비교 | **1. 기존 동화 생성 앱/웹 어플리케이션 (AI스토리교실, Dreammoven)**<br><br>[단순 줄거리 생성]<br>- 초기 단계에서 전체 줄거리를 한 번에 생성하고, 사용자가 결과를 직접 수정·보완하는 방식<br>- **장점**: 빠른 줄거리 생성 가능<br>- **단점**: 아이와의 상호작용이 전혀 없으며, 줄거리를 변경하려면 전체를 다시 생성하거나 사용자가 수동으로 수정해야 함<br><br>[단순 프롬프트 기반 이미지 생성]<br>- DALL·E 등 모델을 사용해 줄거리에 기반한 단순 프롬프트 입력을 통해 완성 이미지를 생성<br>- **장점**: 신속한 이미지 생성, 다양한 캐릭터 표현 가능<br>- **단점**: 아이가 원하는 구도·의상·외형을 반영하기 어려움. 원하는 이미지를 얻기 위해 추가 API 호출 필요 → 불필요한 토큰 소모<br><br>**2. 본 과제의 장점 및 차별성**<br><br>[질의응답 기반 줄거리 생성]<br>- AI가 이전 줄거리를 바탕으로 질문을 제시하고, 아이의 응답을 반영하여 다음 줄거리를 생성<br><br>예시:<br>*질의*: “소연이는 할머니 다락방에서 반짝이는 상자를 발견했어요. 소연이는 무엇을 발견했을까요?”<br>*응답*: “물감 상자”<br>*줄거리 생성*: “요정은 말했어요. ‘이 물감들은 계절의 마법을 지녔단다. 하나를 펼칠 때마다 봄, 여름, 가을, 겨울이 펼쳐지지.’ 소연이가 초록 물감을 칠하자 방 안에 봄바람이 불어왔어요.”<br><br>[아이의 그림 기반 Scribble 이미지 생성]<br>- 아이가 직접 그린 그림 업로드 → ControlNet이 스케치 선 인식 → Stable Diffusion + LoRA가 배경과 캐릭터를 보강하여 최종 이미지 생성<br><br>[음성 복제 기능]<br>- 가족 구성원의 목소리를 AI 나레이션으로 등록 가능<br>- 짱구, 뽀로로, 아기상어, 케데헌, 티니핑 등 캐릭터 음성을 흉내내는 기능 제공<br><br>[이스터에그 기능]<br>- 완성된 그림 내 객체를 YOLOv8으로 인식 후 좌표 반환 → 클릭 시 숨겨진 음성 메시지 팝업<br>- ‘사랑의 편지’ 기능: 열람 가능 시간 및 알림 기능 지원 → 특정 시간 이후 부모 메시지 확인 가능<br><br>[관리자 전용 인터페이스]<br>- 자녀별 동화 관리 기능 제공 (삽입 여부 조회, 마지막 편지/음성 메시지 확인 등)<br>- 이스터에그 페이지 접근 권한 관리자 전용<br>- 삽입 및 열람 시간 설정 지원<br>- 넷플릭스형 구조 도입 → 다자녀 및 교육자 맞춤형 관리 최적화 |  |
| (3) 제안 내용 | 최근 연구에 따르면, 질문 유도 기법을 적용한 인터랙티브 동화 콘텐츠는 유아의 창의성과 언어 표현력 향상에 긍정적인 효과를 보였다.<br><br>단순히 이야기를 수동적으로 듣는 것이 아니라, 아이가 스스로 질문에 답하거나 자신의 경험을 그림, 역할극, 이야기 형태로 표현할 때 학습 효과가 더 크게 나타났으며, 이러한 과정은 부모-자녀 간의 상호작용을 촉진하고 나아가 사회·정서 학습(SEL) 역량을 자기 삶과 연결하는 데도 효과적이었다.<br><br>특히, 이야기를 그림으로 재구성하는 활동은 유아의 상상력과 창의성, 언어 표현 능력을 동시에 자극하는 것으로 확인되었다. 단순히 그림만 그리는 것보다 스토리텔링과 결합했을 때 아이가 사고를 조직화하고 자기 생각을 구체적으로 표현하는 능력이 더욱 발달하였다. 이는 본 서비스의 핵심인 “아이의 그림을 캐릭터화해 동화로 확장하는 경험”과 직결된다.<br><br>아울러, 가정과 유치원의 교육이 연계될 때 학습 효과와 지속성이 높아진다는 점도 여러 연구에서 강조된다. 부모가 교육 과정에 적극적으로 참여하면 아이의 이해와 몰입이 깊어지고, 학습 경험이 일상 속에서 반복·강화되어 발달 효과가 배가된다.<br><br>따라서 부모가 자녀와 함께 동화를 만들고 이야기를 나누는 과정은 단순한 놀이를 넘어 정서적/인지적 발달을 동시에 촉진하는 학습 경험이 된다.<br><br>이러한 연구 결과를 바탕으로, 본 프로젝트는 AI 기반 동화 생성 및 대화형 학습 시스템을 제안한다.<br><br>[해결책]<br>1. AI 동화 생성 및 줄거리 확장<br>- 아이의 이름, 성격, 관심사 등을 반영하여 AI가 맞춤형 동화를 자동 생성<br>- 아이가 직접 그린 그림을 캐릭터로 변환하고, 줄거리 전개 과정에 반영<br>2. 실시간 인터랙티브 대화 시스템<br>- 주인공, 장르, 이전 Scene, 아이의 답변을 기반으로 아이와 자연스러운 질의응답이 오가는 구조<br>- 아이가 대답한 내용이 이야기 전개에 반영되어 능동적 학습 경험을 유도<br>3. 부모-자녀 상호작용 강화<br>- 부모가 자녀와 함께 동화를 완성하고, 숨겨진 메세지나 음성 내레이션을 통해 특별한 교감 제공 |  |
| (4) 기대효과 및 의의 | **1. 스토리텔링 기반 활동 효과**<br>- 연구 결과, 유아가 스토리텔링과 그림 활동을 함께 할 때 창의성과 언어 표현력이 크게 향상됨<br>- 단순히 이야기를 듣는 수동적 학습이 아닌, 아이 스스로 이야기를 만드는 경험을 통해 자기 생각을 구조화하고 표현할 수 있는 능력이 강화됨<br><br>**2. 능동적 학습 경험**<br>- 그림, 역할극, 이야기 등 다양한 방식으로 자신의 경험을 표현하면서 몰입도가 높아짐<br>- 이러한 과정은 아이가 단순 지식 습득을 넘어, 실제 상황에서 적용 가능한 적극적 성찰을 가능하게 함<br><br>**3. 부모-자녀 대화 심화**<br>- 부모와 함께 동화를 만들고 이야기를 나누는 과정에서 대화가 더 깊고 풍부해짐<br>- 아이는 부모와의 상호작용을 통해 사회적·정서적 개념을 자연스럽게 배우고, 이를 자기 삶에 연결할 수 있음<br><br>**4.
